# 📋 My Task Manager

> A personal React JS Task Manager with auto-save, filters, and priority system.

---

## ✨ Features

- ➕ **Add Tasks** — with title and priority level
- ✏️ **Edit Tasks** — update anytime
- 🗑️ **Delete Tasks** — remove instantly
- ✅ **Complete Toggle** — mark tasks done with checkbox
- 💾 **Auto Save** — data stays even after browser closes
- 🔍 **Filter by Status** — All / Pending / Done
- 🎯 **Filter by Priority** — All / High / Medium / Low
- 🧹 **Clear Completed** — remove all done tasks at once

---

## 💡 Key Concepts

- `localStorage` — saves tasks in browser permanently
- `useEffect` — triggers save whenever tasks change
- Filter logic — both status and priority filters work together
- `disabled` prop — edit blocked on completed tasks
